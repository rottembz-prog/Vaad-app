# מנהל תקציבים — ועד הורים

אפליקציית ניהול תקציב לועד הורים. פועלת ישירות בדפדפן, ללא התקנה.

## פתיחה

פתח את הקובץ `index.html` בדפדפן, או השתמש ב-GitHub Pages.

## תכונות

- **תקציב** — ניהול הכנסות והוצאות, מיון לפי תאריך
- **גבייה** — מעקב תשלומי הורים
- **מזומן** — יומן קופה + הוצאות אוטומטיות מהתקציב
- **רעיונות** — מאגר מתנות וספקים
- **לפי קופה** — פירוט יתרות לפי נציג, פיוס מול סה"כ הקופה
- **דו"ח** — ייצוא דו"ח מלא להדפסה

## טכנולוגיה

קובץ HTML יחיד. React 18 + Babel Standalone. אין צורך בבנייה (no build step).
נתונים נשמרים ב-localStorage של הדפדפן.

## GitHub Pages

1. עבור ל-Settings → Pages בריפוזיטורי
2. בחר Branch: `main`, Folder: `/ (root)`
3. שמור — האפליקציה תהיה זמינה בכתובת `https://<username>.github.io/<repo>/`
